## MemPalace

The memory subsystem (`src/mempalace/*`) is a TypeScript port of the core
concepts from MemPalace — wings, rooms, drawers, tunnels, and the knowledge
graph data model. The original implementation is Python; this is a clean-room
reimplementation that preserves the data model and naming conventions.

- Repository: https://github.com/MemPalace/mempalace
